# PostGIS Spatial Analysis

This project focuses on spatial data analysis using PostgreSQL and the PostGIS extension.

## Objective

The goal of this project was to practice geospatial data analysis using SQL, including spatial queries, JOINs, and geometry-based operations.

## Technologies

- SQL
- PostgreSQL
- PostGIS

## Key Features

- Spatial data querying using PostGIS functions
- Geometry operations (e.g. ST_Intersects, ST_Area, ST_DWithin)
- Spatial joins between geographic datasets
- Filtering data using geographic conditions

## Example Queries

See the `/sql` folder for full examples, including:
- Spatial filtering by geometry
- Intersection analysis
- Distance-based analysis using spatial functions

## Skills Demonstrated

- Advanced SQL querying
- Geospatial data analysis
- PostGIS functions usage
